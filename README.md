# Dự án Lập Trình Mobile

## Giới thiệu

Dự án này được xây dựng để thực hành và triển khai các kỹ thuật lập trình di động trên nền tảng Android/iOS. Mục tiêu chính của dự án là học và áp dụng các kiến thức cơ bản và nâng cao về lập trình di động, bao gồm việc phát triển giao diện người dùng, xử lý dữ liệu, và tích hợp với các dịch vụ web.

## Công nghệ sử dụng

- **Ngôn ngữ lập trình**: Kotlin (Android) / Swift (iOS)
- **Nền tảng phát triển**: Android Studio / Xcode
- **Công cụ quản lý dự án**: Git
- **Công cụ kiểm tra**: Unit Testing, UI Testing
- **Thư viện và Frameworks**: Retrofit, Glide, Firebase, etc.
- **Cơ sở dữ liệu**: SQLite / Firebase Realtime Database
- **API**: RESTful API

## Mục tiêu

- Xây dựng ứng dụng di động sử dụng các framework phổ biến như Android SDK, Jetpack Compose (cho Android), hoặc SwiftUI (cho iOS).
- Áp dụng kiến thức về hoạt động đa nhiệm, lưu trữ dữ liệu, và xử lý giao diện người dùng (UI).
- Tích hợp các dịch vụ web và API bên ngoài để lấy dữ liệu từ server.
